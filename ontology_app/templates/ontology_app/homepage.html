PyComplex

Acesse sua ontologia em formato JSON

PyComplex é um serviço online para lidar com ontologias. Esse serviço possibilita a publicação de ontologias na internet. Uma vez publicada a ontologia, será possível acessar os recursos no formato JSON

Publique sua ontologia
Acesse o conteúdo da ontologia pelo seguintes endpoints:
/subclasses

O endpoint /subclasses retorna um array com as subclasses de uma classe específica

/subclasses/{{tutorial.slug}}/Jogador/
[ "Jogador", "Explorador", "Sobrevivente", "Empreendedor", "Conquistador", "Audaz", "Estrategista", "Socializador" ]
/tree

O endpoint /tree retorna um array com a árvore de uma classes

/tree/{{tutorial.slug}}/Jogador/
{ Jogador: [ "Jogador", "Explorador", "Sobrevivente", "Empreendedor", "Conquistador", "Audaz", "Estrategista", "Socializador" ], Explorador: [ "Explorador" ], Sobrevivente: [ "Sobrevivente", "Empreendedor", "Conquistador", "Audaz", "Estrategista" ], Empreendedor: [ "Empreendedor", "Conquistador", "Audaz" ], Conquistador: [ "Conquistador", "Audaz" ], Audaz: [ "Audaz" ], Estrategista: [ "Estrategista", "Conquistador", "Audaz" ], Socializador: [ "Socializador" ] }
/instances

O endpoint /instances retorna um array com as instâncias de uma classe específica

/instances/{{tutorial.slug}}/Jogador/
[ "ana", "julia", "marcos", "mateus" ]
/instance

O endpoint /instance retorna um array com os Types e Properties de uma instância específica

/instance/{{tutorial.slug}}/julia/
[ "Jogador", "Audaz", "Explorador", "Tem_habilidade Criatividade", "Tem_habilidade Discernimento", "Tem_habilidade Noções", "Tem_habilidade Raciocínio_lógico" ]
